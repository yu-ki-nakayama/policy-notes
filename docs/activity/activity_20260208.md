## 得られた学び
1. **国政選挙は「規模」よりも「同時並行性」が難しい**
   事務所運営、対外対応、イベント準備、屋外作業などが同時に走り、優先順位付けと情報共有が質を左右する。個々の作業は単純でも、全体最適を保つには段取りと連携が不可欠だと実感した。

2. **「ルール遵守」と「現場のスピード」を両立させる設計が重要**
   選挙は社会的な信頼の上に成り立つため、運営側には一層の慎重さが求められる。現場ではスピード感が必要な一方、判断に迷う場面ほど確認・相談の導線（誰に聞くか、何を根拠に決めるか）が重要だった。

3. **人脈というより「実務知の獲得」：キーパーソンとの接点が学びを加速する**
   関係者との対話を通じて、地域課題の肌感覚や、運営上の工夫（分担、引き継ぎ、意思決定の仕方）を学べた。現場で信頼される人が持つ暗黙知は、文献だけでは得がたい。

4. **自分の課題：報連相の粒度と体力配分**
   短期決戦では、状況が変わるたびに「結論→理由→次アクション」を短く共有することが効く。自分は疲労が溜まると判断が遅くなる傾向があったため、体力配分と、迷ったら早めに相談する習慣が課題だと分かった。
