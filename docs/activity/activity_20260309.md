# 2026/3/9 議員事務所への講演企画書提出

> 本稿は活動の振り返りを目的とした記録であり、内部情報等に配慮して記述している。

## 基本情報
- 期間：2026/3/9
- 立場：学生団体（[右合の衆](https://www.ugonoshu.org/ja)）メンバー
- 活動場所：議員会館

## 目的
右合の衆として企画している講演会について、関係先に正式に提案を行い、今後の実施可能性につなげることを目的とした。

## 活動内容（公開可能範囲）
- 右合の衆メンバー4名で、[高市早苗内閣総理大臣](https://www.kantei.go.jp/jp/105/meibo/index.html)事務所および[片山さつき財務大臣](https://www.mof.go.jp/about_mof/introduction/personnel/index.html)事務所を訪問した。
- 五月祭や駒場祭などでの講演を想定した企画書を持参し、秘書の方に提出した。
- 本人に直接お会いすることはできなかったが、自分で作成した企画書を正式に受け取っていただくことができた。

## 感想
- 今回の活動を通じて特段大きな学びがあったというよりも、議員会館に実際に足を運び、企画書を提出する経験そのものに意義を感じた。
- とくに、自分で作成した文書を持って政治の現場に赴くことは、素直に良い経験だった。
- 議員会館に比較的身近に出入りできること自体にも、個人的には嬉しさがあった。

## 今後
- 今後は先方の都合次第ではあるが、五月祭や駒場祭、あるいはその他の日程での講演実施に向けて調整が進む可能性がある。
- あわせて、小泉防衛大臣に対しても同様の講演依頼メールを送信する予定である。
